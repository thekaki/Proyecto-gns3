Realizacion de bgp en gns3

La instalacion de gns3 la hemos realizado en debian atraves de repositorios. En primer lugar hemos introducido los repositorios y luego lo hemos instalado. comandos usados: sudo apt update, sudo apt install dirmngr, (repositorio a añadir) sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys F88F6D313016330404F710FC9A2FD067A2E3EF7B - echo "deb http://ppa.launchpad.net/gns3/ppa/ubuntu xenial main" | sudo tee /etc/apt/sources.list.d/gns3.list, sudo apt update y sudo apt install gns3-gui
